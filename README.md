# LinkBox

LinkBox is a personal bookmark manager for saving development references,
documentation, tutorials, troubleshooting links, and study notes.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- Vercel

## Getting Started

Copy the environment file and add your Supabase project values:

```bash
cp .env.example .env.local
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
